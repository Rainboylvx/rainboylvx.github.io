    环境

    rustup (推荐，跨平台)

    1curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

    安装完成后：

    1rustup update      # 更新
2rustc --version    # 查看版本
3cargo --version

    Arch Linux

    1# 方式一：通过 rustup (推荐)
2sudo pacman -S rustup
3rustup default stable
4
5# 方式二：直接安装系统包
6sudo pacman -S rust

    Ubuntu 24.04

    1# 方式一：通过 rustup (推荐)
2sudo apt install curl
3curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
4
5# 方式二：通过 apt
6sudo apt install rustc cargo
