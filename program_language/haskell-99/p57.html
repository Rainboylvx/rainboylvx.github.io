P57 - 构造二叉搜索树

Binary search trees

官方模块：Problems.P57　核心函数：construct、addedTo

← P56 对称二叉树 | P58 对称平衡树 →

题目描述

按输入顺序把元素插入二叉搜索树。小于当前节点的值进入左子树，其余值进入右子树，因此重复值也会保留。

实现

1addedTo :: Ord a => a -> Tree a -> Tree a
2addedTo x Empty = leaf x
3addedTo x (Branch y left right)
4  | x < y     = Branch y (addedTo x left) right
5  | otherwise = Branch y left (addedTo x right)
6
7construct :: Ord a => [a] -> Tree a
8construct = foldl (flip addedTo) Empty

foldl 的累加器是当前树，而 addedTo 的参数顺序是“值在前、树在后”，所以需要 flip。

测试

1>>> construct [3,2,5,7,1]
2Branch 3 (Branch 2 (Branch 1 Empty Empty) Empty)
3         (Branch 5 Empty (Branch 7 Empty Empty))
4>>> symmetric (construct [5,3,18,1,4,12,21])
5True

平均插入成本为 O(log⁡n)O(\log n)O(logn)，但有序输入会退化为链，总构造时间达到 O(n2)O(n^2)O(n2)。
