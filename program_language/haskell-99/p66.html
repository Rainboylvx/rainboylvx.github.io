P66 - 二叉树紧凑布局

Compact binary tree layout

官方模块：Problems.P66　核心函数：layoutCompact

← P65 层级等距布局 | P67 括号字符串表示 →

核心思想

每棵子树记录各深度的 (最左位置, 最右位置) 轮廓。合并左右子树时，寻找最小距离 ddd，使左轮廓右边界平移 -d 后，严格位于右轮廓左边界平移 d 的左侧。

实现

 1type Bounds = [(Int, Int)]
 2data Relative a = REmpty | RNode a Int Bounds (Relative a) (Relative a)
 3
 4layoutCompact :: Tree a -> Tree (a, (Int, Int))
 5layoutCompact tree = absolute relative (1 - leftMost relative) 1
 6  where relative = compact tree
 7
 8compact :: Tree a -> Relative a
 9compact Empty = REmpty
10compact (Branch value left right) =
11  RNode value 0 bounds (move (-distance) l) (move distance r)
12  where
13    l = compact left
14    r = compact right
15    distance = safeDistance (contour l) (contour r)
16    bounds = merge (shiftBounds (-distance) (contour l))
17                   (shiftBounds distance (contour r))
18
19contour REmpty                = []
20contour (RNode _ _ bs _ _)    = (0,0) : bs
21
22move _ REmpty = REmpty
23move d (RNode x p bs l r) = RNode x (p+d) bs l r
24
25shiftBounds d = map (\(l,r) -> (l+d,r+d))
26
27merge [] ys = ys
28merge xs [] = xs
29merge ((l,_):xs) ((_,r):ys) = (l,r) : merge xs ys
30
31safeDistance left right = head
32  [d | d <- [1..], and (zipWith (separated d) left right)]
33  where separated d (_,lr) (rl,_) = lr - d < rl + d
34
35leftMost REmpty = 0
36leftMost (RNode _ p bs _ _) = minimum (p : map fst bs)
37
38absolute REmpty _ _ = Empty
39absolute (RNode value offset _ left right) parentX depth =
40  Branch (value,(x,depth))
41    (absolute left x (depth+1))
42    (absolute right x (depth+1))
43  where x = parentX + offset
轮廓而非节点总宽度

两棵子树能否靠近，取决于它们在每一层的相向边界。只比较整体宽度会错过可以交错利用的空白。

验证要点

  • 每个子节点仍在父节点相应方向；
  • 同一坐标没有两个节点；
  • 任意再缩小一次左右间距都会在某层发生碰撞。
