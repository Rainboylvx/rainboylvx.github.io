P65 - 二叉树层级等距布局

Binary tree layout with constant distance at each level

官方模块：Problems.P65　核心函数：layoutLevelConstant

← P64 中序布局 | P66 紧凑布局 →

思路

同一深度的父子水平距离相同，并且每深入一层距离减半。先以根坐标 0 生成布局，再整体右移，让最左节点位于 x=1x=1x=1。

实现

 1layoutLevelConstant :: Tree a -> Tree (a, (Int, Int))
 2layoutLevelConstant tree = shift (1 - minimumX raw) raw
 3  where
 4    gap = 2 ^ max 0 (treeHeight tree - 2)
 5    raw = place tree 0 1 gap
 6
 7place Empty _ _ _ = Empty
 8place (Branch value left right) x depth gap =
 9  Branch (value, (x, depth))
10    (place left  (x - gap) (depth + 1) (gap `div` 2))
11    (place right (x + gap) (depth + 1) (gap `div` 2))
12
13minimumX Empty = 0
14minimumX (Branch (_, (x,_)) left right) =
15  minimum [x, minimumX left, minimumX right]
16
17shift _ Empty = Empty
18shift amount (Branch (value,(x,y)) left right) =
19  Branch (value,(x + amount,y)) (shift amount left) (shift amount right)

测试

1>>> layoutLevelConstant (Branch 'a' (leaf 'b') (leaf 'c'))
2Branch ('a',(2,1))
3  (Branch ('b',(1,2)) Empty Empty)
4  (Branch ('c',(3,2)) Empty Empty)

这套规则保证同层间距规律统一，但可能留下较多空白；P66 会根据子树轮廓进一步压缩。
