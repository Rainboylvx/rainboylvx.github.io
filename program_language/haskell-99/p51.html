P51 - 三重复纠错码

Error-correcting codes

官方模块：Problems.P51　核心函数：errorCorrectingEncode、errorCorrectingDecode

← P50 Huffman 编码 | P52 合取范式 →

题目描述

用长度为 3 的重复码编码布尔位：False 变为三个 False，True 同理。解码时对每组三位多数表决，因此每组最多一个错误仍可恢复。

实现

1errorCorrectingEncode :: [Bool] -> [Bool]
2errorCorrectingEncode = concatMap (replicate 3)
3
4errorCorrectingDecode :: [Bool] -> [Bool]
5errorCorrectingDecode [] = []
6errorCorrectingDecode xs = majority chunk : errorCorrectingDecode rest
7  where
8    (chunk, rest) = splitAt 3 xs
9    majority bits = length (filter id bits) * 2 > length bits

随机破坏辅助函数

 1import Data.List (nub, sort)
 2import System.Random (RandomGen, randomRs)
 3
 4corrupt :: RandomGen g => g -> Int -> [Bool] -> [Bool]
 5corrupt gen n bits = flipAt 0 positions bits
 6  where
 7    count = min (max 0 n) (length bits)
 8    positions = sort . take count . nub $
 9      randomRs (0, length bits - 1) gen
10    flipAt _ _ [] = []
11    flipAt i ps@(p:rest) (b:bs)
12      | i == p    = not b : flipAt (i + 1) rest bs
13      | otherwise = b : flipAt (i + 1) ps bs
14    flipAt _ [] remaining = remaining

测试

1>>> errorCorrectingEncode [True,False]
2[True,True,True,False,False,False]
3>>> errorCorrectingDecode [True,False,True,False,False,True]
4[True,False]
5>>> errorCorrectingDecode (errorCorrectingEncode [True,False,True])
6[True,False,True]

代价是数据膨胀三倍，收益是每个三位块能纠正一个翻转错误。
