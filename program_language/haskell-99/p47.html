P47 - 用 NAND 构造通用逻辑电路

Universal logic gates

官方模块：Problems.P47　核心函数：evaluateCircuit、buildCircuit

← P46 二元逻辑真值表 | P48 N 元真值表 →

电路表示

(-1)、(-2) 分别代表两个输入。列表中第 iii 对 (a,b) 表示第 iii 个 NAND 门读取编号 a、b，最后一个门就是电路输出。

 1evaluateCircuit :: [(Int, Int)] -> Bool -> Bool -> Bool
 2evaluateCircuit gates x y = last outputs
 3  where
 4    inputs = [(-1, x), (-2, y)]
 5    outputs = build 1 inputs gates
 6    build _ env [] = map snd (drop 2 (reverse env))
 7    build i env ((a,b):gs) =
 8      let value = not (lookupValue a env && lookupValue b env)
 9      in build (i + 1) ((i,value):env) gs
10    lookupValue k env = case lookup k env of
11      Just v  -> v
12      Nothing -> error "evaluateCircuit: invalid gate reference"

从真值表构造电路

NAND 可以实现非、与、或。下面先构造四个最小项，再把函数为真的项析取；最后加两次非，使输出一定是最后一个门。

 1data Builder = Builder Int [(Int, Int)]
 2
 3emit :: Int -> Int -> Builder -> (Int, Builder)
 4emit a b (Builder next gs) = (next, Builder (next + 1) (gs ++ [(a,b)]))
 5
 6notGate a st = emit a a st
 7
 8andGate a b st0 =
 9  let (t, st1) = emit a b st0
10  in notGate t st1
11
12orGate a b st0 =
13  let (na, st1) = notGate a st0
14      (nb, st2) = notGate b st1
15  in emit na nb st2
16
17buildCircuit :: (Bool -> Bool -> Bool) -> [(Int, Int)]
18buildCircuit f = gates finalState
19  where
20    st0 = Builder 1 []
21    (nx, st1) = notGate (-1) st0
22    (ny, st2) = notGate (-2) st1
23    (m00, st3) = andGate nx ny st2
24    (m01, st4) = andGate nx (-2) st3
25    (m10, st5) = andGate (-1) ny st4
26    (m11, st6) = andGate (-1) (-2) st5
27    selected = [m | ((a,b),m) <- zip assignments [m00,m01,m10,m11], f a b]
28    assignments = [(False,False),(False,True),(True,False),(True,True)]
29    (raw, st7) = combine selected st6
30    (negated, st8) = notGate raw st7
31    (_, finalState) = notGate negated st8
32
33    combine [] st = let (t, st') = emit (-1) nx st in notGate t st'
34    combine (g:gs) st = foldl step (g, st) gs
35    step (acc, st) g = orGate acc g st
36
37    gates (Builder _ gs) = gs
