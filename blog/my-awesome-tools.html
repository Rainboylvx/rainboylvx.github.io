1 Linux Tool

  • dialog - 用于在 shell 脚本中创建文本用户界面的工具，支持对话框、菜单、进度条等交互元素
  • gum：一个由 Charmbracelet 开发的命令行工具，用于创建漂亮的交互式 Shell 脚本，支持输入、选择、确认等 UI 组件，适合增强终端用户体验。
  • sunlei awesome-tools - 一个收集各种实用工具和资源的精选列表，涵盖开发、设计、效率等多个领域

3 c++

  • nlohmann/json - 现代 C++ 的 JSON 库，提供直观的 API 和强大的功能，支持序列化、反序列化和操作 JSON 数据

4 Js

  • 🎮 kaplay - 一个 JavaScript/TypeScript 游戏开发库，提供简单易用的 API 和丰富的游戏开发功能
  • Manim.js - 基于 JavaScript (p5.js) 的数学动画引擎，复刻 3Blue1Brown 的动画风格，用于创建数学可视化内容
